### What I learned

This was my first advanced usage of the grid-breakout layout approach. I used a total of 6 width tiers (full-width, ulta-wide, breakout, popout, content and narrow). The setup was abit difficult but I got a good grasp on how it works now

### Continued development

There are some things that i can improve on the grid layout. Especially for the footer section of this project. I feel like i had to override alot of things to achieve the desired layout.

### Useful resources

- [Layout Breakouts with CSS Grid](https://ryanmulligan.dev/blog/layout-breakouts/) - Blog post about the layout approach I used
